# I18N Translate — Action 文档

## 总览

插件注册了 1 个顶级菜单组（`I18nEditorPopupGroup`），挂载到 `EditorPopupMenu` 和 `ProjectViewPopupMenu`。组内根据当前文件上下文（非 i18n 文件 / 未绑定文件 / 源文件 / 目标文件）动态组装子项。

## 1. Action 一览

| ID | 类 | 类型 | 右键触发条件 |
|----|----|------|-------------|

| `I18nTranslate.AddTranslateOne` | `AddKeyAndTranslateOneAction` | action | 同上 |

| `I18nTranslate.FileAllKeys` | `TranslateEntireSourceFileAction` | action | 源 i18n 文件 |

| `I18nTranslate.ReplaceTargetFilePath` | `ReplaceTargetFilePathAction` | action | 已绑定目标的 i18n 文件 |

## 2. 菜单组逻辑

入口：`I18nEditorPopupGroup.getChildren(e)`

```
getChildren(e)
├── if (非 i18n 文件 && 有选词)
│   ├── 添加到源
│   ├── 添加到源并翻译到所有目标
│   └── 添加到源并翻译到目标
│
├── if (未绑定 i18n 文件)
│   ├── 添加文件到源 (若未配源路径)
│   └── 添加文件到目标
│
├── if (已绑定的目标 i18n 文件)
│   └── 替换文件到目标
│
├── if (源 i18n 文件)
│   ├── if (编辑器 && 光标在 key 上 && 有目标)
│   │   ├── 翻译到目标 → [子菜单: 各目标语种]
│   │   └── 翻译到所有目标
│   ├── if (编辑器 && 选区含有效 key && 有目标)
│   │   ├── 选中keys翻译到目标 → [子菜单: 各目标语种]
│   │   └── 选中keys翻译到所有目标
│   └── 翻译文件所有 key 到目标
```

每个 `Separator` 在不同功能组之间插入。

## 3. 各 Action 详解

### 3.1 从源码选取文本写入

**触发条件**：在非 i18n 文件（如 `.java`, `.kt`, `.js`）中选中文本或光标停在某个词上。

#### 3.1.1 AddKeyToSourceAction

| 项目 | 说明 |
|------|------|
| 入口 | `update()` → `isEnabledAndVisible = true` |
| 流程 | `AddKeyDialog(SOURCE_ONLY)` → 确认覆盖 → `appendSourceKey` |
| 结果 | 仅写入源文件，不翻译 |

#### 3.1.2 AddKeyAndTranslateAllAction

| 项目 | 说明 |
|------|------|
| 入口 | `update()` → `isEnabledAndVisible = true` |
| 流程 | `AddKeyDialog(ALL_TARGETS)` → 确认覆盖 → `appendSourceKey` + 对所有目标执行 `translateToTargets` |
| 结果 | 写入源文件 + 翻译到所有已配目标 |

#### 3.1.3 AddKeyAndTranslateOneAction

| 项目 | 说明 |
|------|------|
| 入口 | `update()` → `isEnabledAndVisible = true` |
| 流程 | `AddKeyDialog(SINGLE_TARGET, targetOptions)` → 选目标 → 确认覆盖 → `appendSourceKey` + 对选定目标 `translateToTargets` |
| 结果 | 写入源文件 + 翻译到选定的一个目标 |

---

### 3.2 源文件中单 key 翻译

**触发条件**：源 i18n 文件，光标放在一个 key 上。

#### 3.2.0 TranslateToTargetDynamicGroup

| 项目 | 说明 |
|------|------|
| 类型 | `ActionGroup(popup=true)` |
| 标签 | `action.translate.to` → "翻译到目标" |
| 子项 | 每个有效目标生成一个 `TranslateKeyToTargetAction` |

#### 3.2.1 TranslateKeyToTargetAction

| 项目 | 说明 |
|------|------|
| 构造 | 接收 `TargetEntry`，`presentation.text = language displayLabelWithCode` |
| 流程 | `detectKey` 解析光标下 key/value → `TranslateKeyConfirmDialog`(可改 value) → 对单个目标 `translateToTargets` |
| 锚点 | `previousKeyInSameFile(orderedKeys, key)` |

#### 3.2.2 TranslateKeyToAllTargetsAction

| 项目 | 说明 |
|------|------|
| 标签 | `action.translate.all.targets` → "翻译到所有目标" |
| 流程 | `detectKey` → `TranslateKeyConfirmDialog(all targets)`(可改 value) → 对所有目标 `translateToTargets` |
| 锚点 | `previousKeyInSameFile(orderedKeys, key)` |

---

### 3.3 源文件中多 key 翻译（批量选中）

**触发条件**：源 i18n 文件，在编辑器中选中多行 key-value 文本。

Key 解析：`I18nKeyDetector.selectedKeyValues(editor, path)` — 从选区文本中直接解析 key 和 value，不依赖磁盘（支持未保存内容）。

#### 3.3.0 TranslateSelectedKeysToTargetDynamicGroup

| 项目 | 说明 |
|------|------|
| 类型 | `ActionGroup(popup=true)` |
| 标签 | `action.translate.selected.to` → "选中keys翻译到目标" |
| 子项 | 每个有效目标生成一个 `TranslateSelectedKeysToTargetAction` |

#### 3.3.1 TranslateSelectedKeysToTargetAction

| 项目 | 说明 |
|------|------|
| 构造 | 接收 `TargetEntry`，`presentation.text = language displayLabelWithCode` |
| 流程 | `selectedKeyValues` → `TranslateSelectedKeysDialog`(确认弹窗，左 table 展示 key-value，右显示目标路径) → 对每个 key 依次 `translateToTargets` |
| 锚点 | `previousKeyInSameFile(orderedKeys, key)` (基于源文件全局排序) |

#### 3.3.2 TranslateSelectedKeysToAllTargetsAction

| 项目 | 说明 |
|------|------|
| 标签 | `action.translate.selected.all` → "选中keys翻译到所有目标" |
| 流程 | `selectedKeyValues` → `TranslateSelectedKeysDialog`(左 table，右显示所有目标路径) → 对每个 key 依次对所有目标 `translateToTargets` |
| 锚点 | 同上 |

---

### 3.4 整文件翻译

#### 3.4.0 TranslateEntireSourceFileAction

| 项目 | 说明 |
|------|------|
| 标签 | `action.file.translate.all` → "翻译文件所有 key 到目标" |
| 触发条件 | 文件为源 i18n 文件（不依赖编辑器光标或选区） |
| 流程 | 读取所有 key → `TranslateFileKeysDialog`(选目标) → 逐 key `translateToTargets` |
| 锚点 | `previousKeyInSameFile` 按源文件顺序 |

---

### 3.5 文件路径绑定

#### 3.5.0 AddSourceFilePathAction

| 项目 | 说明 |
|------|------|
| 标签 | `action.add.path.source` → "添加文件到源" |
| 触发条件 | 未绑定 i18n 文件，且未配源路径 |
| 流程 | `AssignI18nFilePathDialog(SOURCE)` → 选语种 → 保存到 `ProjectI18nConfigService` |

#### 3.5.1 AddTargetFilePathAction

| 项目 | 说明 |
|------|------|
| 标签 | `action.add.path.target` → "添加文件到目标" |
| 触发条件 | 未绑定 i18n 文件 |
| 流程 | `AssignI18nFilePathDialog(TARGET)` → 选已有目标行/新建语种 → 保存 |

#### 3.5.2 ReplaceTargetFilePathAction

| 项目 | 说明 |
|------|------|
| 标签 | `action.replace.target.path` → "替换文件到目标" |
| 触发条件 | 当前文件已绑定为目标文件 |
| 流程 | `AssignI18nFilePathDialog(TARGET, replaceTargetFlow=true)` → 重新指定语种 → 更新目标列表（旧语种路径清空） |

## 4. 共用工具

### I18nActionSupport

| 方法 | 说明 |
|------|------|
| `updateThread()` | 返回 `ActionUpdateThread.BGT` |
| `selectedKeyText(event)` | 获取编辑器选区文本或光标下单词 |
| `ensureSourceConfigured(project)` | 检查源路径是否已配，否则弹警告 |
| `ensureTargetsConfigured(project)` | 检查是否有有效目标，否则弹警告 |
| `confirmSourceKeyOverwrite(project, key)` | 源文件已存在该 key 时询问是否覆盖 |

### I18nKeyDetector

| 方法 | 说明 |
|------|------|
| `detectKey(editor, content, path)` | 光标下的 key + value（读盘+编辑器位置） |
| `selectedTextOrWord(editor)` | 选区文本 / 光标词 |
| `selectedKeyValues(editor, path)` | 选区中所有 key-value 对（直接从选区文本解析，不读盘） |
| `selectedKeys(editor, path)` | 仅 key 列表（委托给 selectedKeyValues） |
| `previousKeyInSameFile(orderedKeys, key)` | 源文件中前一 key 名（用作目标文件插入锚点） |

### I18nTranslateWorkflow

| 方法 | 说明 |
|------|------|
| `appendSourceKey(project, sourcePath, key, value)` | 向源文件追加/更新 key |
| `sourceContainsKey(project, sourcePath, key)` | 检查源文件中是否存在 key |
| `readSourceValue(project, sourcePath, key)` | 从磁盘读取 key 的 value |
| `orderedKeys(project, sourcePath)` | 源文件 key 顺序列表 |
| `translateToTargets(project, text, sourceLang, targets, anchor, key, log)` | 翻译并写入一个或多个目标 |
| `translateKeysSequential(project, sourcePath, sourceLang, keys, targets, log)` | 按序批量翻译（读盘取值） |
